ID e CLASS

ID

O ID funciona de maneira como um CPF. ASsim como só posso ter um numero de CPF, cada elemento só pode ter um valor de ID. O CPF é unico, ou seja, niguem pode ter um CPF igual ao meu. Isso significa que o ID também é unico, entao nao é possivel repetir o mesmo valor

CLASS

O CLASS funciona de maneira oposta do ID. Diferente do ID em que você só pode ter um valor, no CLASS voce pode ter varios valores. Também diferente di ID, em que nao pode repetir um valo, no CLASS voce pode utilizar o mesmo valor para varios elementos.
